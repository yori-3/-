# Personal Site

頼 毅（より つよし）さん向けに作成した、仕事用のシンプルな個人ホームページです。

## ファイル構成

- `index.html`: ページ本体
- `styles.css`: デザイン
- `script.js`: スクロール表示アニメーション
- `.nojekyll`: GitHub Pages でそのまま公開しやすくする設定

## 先に変更するとよい場所

1. 必要なら「業務内容」や「プロフィール」を実情に合わせて調整
2. 写真や会社情報を追加したい場合は `index.html` を編集

## ローカルで見る

`personal-site/index.html` をブラウザで開くと確認できます。

## 公開する方法

### 1. いちばん簡単: Netlify Drop

1. `personal-site` フォルダを zip にする
2. [Netlify Drop](https://app.netlify.com/drop) を開く
3. zip をドラッグ&ドロップする
4. すぐに公開URLが発行されます

### 2. GitHub Pages

1. GitHub で新しいリポジトリを作成
2. `personal-site` の中身をアップロード
3. リポジトリの `Settings > Pages` を開く
4. Branch を `main`、Folder を `/ (root)` に設定
5. 数分で公開されます

独自ドメインを使いたい場合は、公開後にドメイン設定を追加できます。
